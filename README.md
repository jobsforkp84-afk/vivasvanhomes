# Vivasvan Homes Website

A modern, responsive website for Vivasvan Homes - a trusted channel partner for premium real estate projects.

## Overview

Vivasvan Homes is a real estate channel partner specializing in premium residential properties. The website showcases their partnership with leading developers and helps potential homebuyers find their dream homes.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Project Showcase**: Highlights partner projects (Abhie Areia and Sanjeevni Advith)
- **Contact Form**: Interactive contact form for inquiries
- **Smooth Navigation**: Fixed navigation with smooth scrolling
- **About Section**: Company information and statistics

## Property Information

- **Price Range**: ₹1.5 Cr - ₹3 Cr
- **Property Types**: 2 BHK, 3 BHK, and 4 BHK apartments
- **Partner Projects**:
  - Abhie Areia (https://abheeprojects.com/)
  - Sanjeevni Advith

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required - it's a static website

## Customization

### Updating Contact Information
Edit the contact section in `index.html` to update phone numbers, email addresses, or location details.

### Adding New Projects
Add new project cards in the projects section of `index.html` following the existing structure.

### Styling
Modify `styles.css` to change colors, fonts, or layout. The CSS uses CSS variables for easy customization:
- `--primary-color`: Main brand color
- `--secondary-color`: Secondary accent color
- `--text-dark`: Dark text color
- `--text-light`: Light text color
